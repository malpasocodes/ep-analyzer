# Data Sources — EP Analyzer

This document describes every data source used in EP Analyzer: where it comes from, what it contains, how it is generated, and known limitations.

---

## Overview

EP Analyzer combines four external data sources into a layered dataset:

1. **College Scorecard** — institution-level graduate earnings
2. **IPEDS** — institution metadata, sector/control/level, county FIPS codes, program completions
3. **Census ACS B20004** — county-level HS graduate median earnings (local benchmark)
4. **State Thresholds 2024** — statewide HS earnings benchmarks (the policy benchmark under the One Big Beautiful Bill Act)

The pipeline merges these into `ep_analysis.parquet` (base) and `ep_analysis_enriched.parquet` (with real county earnings). The backend auto-detects and prefers the enriched version at startup.

## External Data Sources

### Census ACS 5-Year Estimates — Table B20004

- **URL**: `https://api.census.gov/data/{year}/acs/acs5`
- **Variable**: `B20004_003E` — Median earnings in the past 12 months for the population 25 years and over with earnings, educational attainment: High school graduate (includes equivalency)
- **Margin of error**: `B20004_003M`
- **Geography**: All U.S. counties (`for=county:*&in=state:*`)
- **Default vintage**: 2023 (= 2019–2023 5-year estimates)
- **API key**: Optional; recommended for higher rate limits. Register free at `https://api.census.gov/data/key_signup.html`
- **Suppression**: The Census suppresses estimates for small populations (returns `-666666666`). The pipeline drops these rows.
- **Age note**: B20004 covers **all ages 25+**, not the 25–34 age band sometimes used in policy contexts. This is the finest pre-tabulated county-level table available from ACS.

### College Scorecard

- **Variables used**: `MD_EARN_WNE_P6` (median earnings 6 years post-enrollment) and `MD_EARN_WNE_P10` (10 years post-enrollment)
- **Population**: Students who received federal financial aid
- **Matching key**: IPEDS UnitID
- **Source file**: `scorecard_earnings.csv`

### IPEDS (Integrated Postsecondary Education Data System)

- **Variables used**: UnitID, institution name, state abbreviation (`STABBR`), sector/control/level, county FIPS code (`FIPS` column in `ipeds_institutions.csv`), program completion counts
- **Source files**: `ipeds_institutions.csv`, `program_counts.parquet`
- **FIPS column**: The raw IPEDS FIPS value is a numeric integer (e.g., `6037` for Los Angeles County); the pipeline zero-pads it to 5 digits.
- **Download**: Available from `https://nces.ed.gov/ipeds/use-the-data`

### State Thresholds 2024

- **File**: `state_thresholds_2024.csv`
- **Content**: 2024 statewide median earnings for high school graduates (50 states + DC + national average), 52 entries total
- **Derivation**: Aggregated from Census ACS at the state level
- **Policy role**: These thresholds are the benchmark used by the One Big Beautiful Bill Act to evaluate whether a program "passes" the earnings premium test. An institution's graduate median earnings must meet or exceed its state's HS graduate median earnings to pass.

## Data Enrichment Pipeline

The pipeline in `backend/app/pipelines/fetch_county_earnings.py` generates the three non-checked-in data files:

```
Step 1: Fetch Census ACS B20004 for all counties
        → data/county_hs_earnings.csv

Step 2: Build UnitID → county FIPS mapping from IPEDS
        → data/institution_county_mapping.csv

Step 3: Merge ep_analysis.parquet with county earnings
        → data/ep_analysis_enriched.parquet
```

**Run the pipeline:**

```bash
# Default: ACS 2023 vintage, no API key
python -m backend.app.pipelines.fetch_county_earnings

# With a specific year and Census API key
python -m backend.app.pipelines.fetch_county_earnings --year 2022 --api-key YOUR_KEY
```

**Prerequisites**: `data/ipeds_institutions.csv` must exist before running Step 2.

## Benchmark Reclassification Logic

The analysis router (`/api/analysis/reclassify`) compares each institution against two benchmarks:

| Benchmark | Source | Notes |
|-----------|--------|-------|
| **State benchmark** | `state_thresholds_2024.csv` via `Threshold` column | Single value per state; the statutory policy threshold |
| **Local benchmark** | `county_hs_earnings.csv` via `county_hs_earnings` column | Real county data when available; otherwise synthetic fallback |

**Synthetic fallback** (`benchmark_source = "synthetic"`): When an institution lacks county data, a synthetic local benchmark is generated by applying a CZ-type multiplier (urban 0.90–1.25×, suburban 0.85–1.10×, rural 0.70–1.00×) plus Gaussian noise. The `inequality` slider (0–1) scales the noise standard deviation from $1,500 to $8,000. This only affects institutions without real county data.

**Reclassification outcomes**:

| Classification | Meaning |
|----------------|---------|
| Pass Both | Earnings ≥ state threshold AND ≥ local benchmark |
| Fail Both | Earnings < state threshold AND < local benchmark |
| Pass Local Only | Earnings ≥ local benchmark but < state threshold (geographic bias: state penalizes low-wage area) |
| Pass State Only | Earnings ≥ state threshold but < local benchmark (local market is unusually high-wage) |

## Coverage and Caveats

### Coverage

- **County earnings coverage**: Approximately 3,204 of the ~3,142 U.S. counties have usable ACS B20004 estimates (some are suppressed for small populations).
- **Institution-to-county match rate**: Approximately 5,998 of 6,429 institutions in the full dataset (93%) can be mapped to a county via IPEDS FIPS; the remainder fall back to synthetic benchmarks. Note: the main dashboard filters to 6,275 institutions in 50 states + DC.
- **Scorecard coverage**: Not all institutions have Scorecard earnings data; records with missing `median_earnings` are excluded from reclassification.

### Known Limitations

1. **ACS age band**: B20004_003E covers all adults 25+, not the 25–34 cohort most comparable to recent graduates. Older workers in the same education band may have higher earnings, making county benchmarks somewhat conservative.
2. **Temporal mismatch**: The default pipeline uses ACS 2023 (2019–2023 5-year) county earnings. Scorecard earnings are from a different cohort window. The state thresholds are labeled 2024 but derived from ACS data of a similar vintage.
3. **Synthetic benchmarks are random**: For institutions without county data, benchmarks are generated deterministically (seed=42) but are not based on real local conditions.
4. **IPEDS FIPS**: The FIPS column in IPEDS refers to the county of the main campus. Branch campuses and online-only institutions may serve students across many counties.
5. **National average**: `state_thresholds_2024.csv` includes a "United States (National)" row (52 entries total); the backend filters to 50 states + DC only (`VALID_STATES`). Institutions in territories (PR, GU, VI, etc.) are excluded from the main dashboard.
